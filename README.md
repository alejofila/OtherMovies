
# Sample demo
The main objective of this demo is to show a list of _popular tv shows_. 
The _data source_ is **TMDB API** the projects showcase some popular topics in **android development world** : 


* Kotlin
* RxJava
* Retrofit
* Picasso
* Constraint Layout, Constraint Layout Animations
* Model View Presenter
* Unit Testing with JUnit and Mockito
* Modular codebase
* Koin as a dependency injection library
